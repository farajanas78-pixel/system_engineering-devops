# 0x04. Loops, conditions and parsing

This directory contains Bash scripts that demonstrate loops, conditional statements, and input parsing.

## Tasks

0. **1-for_best_school** - For loop displaying "Best School" 10 times
1. **2-while_best_school** - While loop displaying "Best School" 10 times
2. **3-until_best_school** - Until loop displaying "Best School" 10 times
3. **4-if_9_say_hi** - While loop with if statement showing "Hi" on 9th iteration
4. **5-4_bad_luck_8_is_your_chance** - While loop with if-elif-else for different messages
5. **6-superstitious_numbers** - While loop with case statement for numbers 1-20
6. **8-for_ls** - For loop processing filenames (extracting part after first dash)
7. **9-to_file_or_not_to_file** - File existence and property checks with if-else
